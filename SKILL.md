## When to Run

- 用于计算LPR利息。
- 对于诉讼环境下的利息计算，一般默认使用LPR计算利息。
- 若借贷起始日期早于LPR首次公布日`2019-08-20`，计算工具将忽略LPR首次公布日之前的利息。对于不被计算的部分利息，应调用其他工具单独计算后与本次结果进行相加。

## Workflow

1. 运行`intcalc update`更新LPR源
2. 尝试获取参数信息，包括`贷款本金`、`起始日期`、`截至日期`、`利率倍率`、`年天数基数`、`是否采取5年期利率`
3. 上下文及相关文件可能已经提供了足够的参数信息，应当先对上下文及文件进行检索。
4. 如上下文及相关文件没有提及，则应当向用户提问以获取。对于默认值或长期约定的习惯值，可不要求用户提供。
5. 无论参数信息来源（上下文/默认值/用户习惯/用户提供），使用命令前都应该对传入的参数向用户明示。
6. 如果是小规格任务（如利息计算），向用户明示参数后可直接开始执行。如果是大规模任务的子任务（如撰写起诉状时需计算利息），向用户明示后应等待用户核对确定，然后再开始执行。
7. 获取足够信息后使用`intcalc`进行计算，具体参数见使用示例。

## 使用示例

### 与用户交互

```
# 示例1
用户> (提供相关文件)
用户> 计算lpr利息

你> (分析文件，获得足够信息；执行指征充分，向用户明示参数信息)
你> 本金为20,000；起始日期为2022-01-02；截止日期未提供，默认今日(2026-04-15)；利率倍率未提供，默认使用1.0；年天数基数未提供，默认使用365天；默认采取1年期利率。
你> intcalc update
你> intcalc lpr 20000 2022-01-02 2026-04-15 -p 1.0 -d 365

# 示例2
用户> 本金1000000，从2024-01-01计至2026-01-01,按LPR4倍、年360天、5年期利率计算lpr。我要知道具体计算细则

你> 按照本金1,000,000；起始日期为2024-01-01；截止日期2026-01-01；利率倍率4.0；年天数基数用360天；采取5年期利率进行计算。
你> intcalc update
你> intcalc --debug lpr 1000000 2024-01-01 2026-01-01 -p 4 -d 360 --five
```

### 更新LPR利率数据

从中国银行获取最新利率数据，确保计算准确性。

```bash
intcalc update
```

### 计算利息

计算指定贷款的利息。

```bash
intcalc lpr [贷款本金] [起始日期] [截至日期(默认今日)] -p [倍率(默认1.0)] -d [年天数基数(默认365)] --five

# 如 2000000，从2020-01-31计至2026-04-16,按LPR4倍、年365天计算、采取5年期利率
intcalc lpr 2000000 2020-01-31 2026-04-16 -p 4 --five
# 输出 2088263.01

# 如 2000000，从2024-12-31计至2026-04-16,按LPR4倍、年365天计算
intcalc lpr 2000000 2024-12-31 2026-04-16 -p 4
# 输出 312767.12

# 如需展示计算细则
intcalc --debug lpr 2000000 2026-02-25 2026-04-16 -p 4
# 输出格式为 分段起始日, 分段结束日, 间隔天数, 利率, 利息
# 2026-02-25, 2026-03-20, 23, 0.03, 15123.29
# 2026-03-20, 2026-04-16, 27, 0.03, 17753.42
# 32876.71
```

## 参数说明

- `[贷款本金]`: 贷款的总金额。
- `[起始日期]`: 计息开始的日期，格式为 `YYYY-MM-DD`。
- `[截至日期]`: 计息结束的日期（可选，默认为当前日期）。
- `-p [倍率]`: LPR利率的倍数（可选，默认为 `1.0`）。
- `-d [年天数基数]`: 计算日利率时的年天数基数（可选，默认为 `365`）。
- `--five`: 是否采取5年期利率(可选，默认为不开启，即默认采取1年期利率)。
- `--debug`: 是否输出具体计算细节（可选，默认不开启）。

## 注意事项
- LPR利率来源: 数据抓取自中国银行贷款市场报价利率页面。
- 算头不算尾: 日期区间为左闭右开，例如`[2024-01-01, 2024-01-03)`计算为2天。
- 分段计算: 计算工具将根据LPR利率调整的时间点进行分段计算。
- 长周期借贷: 若起始日期早于LPR首次公布日`2019-08-20`，则从公布日开始计息，之前部分将被计算器忽略。
- 计算精度: 分段计算精度为`f64`，累加后对结果四舍五入保留两位小数
