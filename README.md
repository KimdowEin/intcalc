## 功能

- Update: 从中国银行网站获取最新LPR利率数据并保存到本地CSV
- Lpr: 以LPR计算贷款利息

## 使用

### 更新LPR利率

```bash
intcalc update
```

### 计算利息

```bash
intcalc lpr [贷款本金] [起始日期] [截至日期(默认今日)] -p [倍率(默认1.0)] -d [日利率基数(默认365)] 

# 如 本金1000000，从2024-01-01计至2026-01-01,按LPR4倍、年360天计算
intcalc lpr 1000000 2024-01-01 2026-01-01 -p 4 -d 360

# 如 2000000，从2024-12-31计至2026-04-16,按LPR4倍、年365天计算
intcalc lpr 2000000 2024-12-31 2026-04-16 -p 4
# 输出 312767.12

# 核对每个分段的计算
intcalc lpr 2000000 2024-12-31 2026-04-16 -p 4 --debug

```

## 注意事项

- LPR利率来源：抓取自[中国银行贷款市场报价利率](https://www.boc.cn/fimarkets/lilv/fd32/201310/t20131031_2591219.html)
- 算头不算尾：[2024-01-01,2024-01-03)为2天
- 分段计算：采取LPR利率分段计算
- 长周期借贷：若起始日期早于该LPR首次公布日，则从公布日开始计息，之前部分忽略。
- 利率选择：根据借贷周期进行计算，大于等于5年的采取5年期利率。计算方法为`日利率基数 x 5`
